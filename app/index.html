Swagger Specification Visual Documentation

This is a prototype of a D3 rendering of the Swagger specification to provide an easy to read visual documentation.
It is developed by Arnaud Lauret, the API Handyman.

This is NOT a rendering of an API Swagger specification (that will come next).

Controls

Zoom in: Mouse scroll up
Zoom out: Mouse scroll down
Move: Mouse drag
Moving mouse over a node shows its documentation
Clicking on a node expand or collapse it

Developed by Arnaud Lauret, the API Handyman
