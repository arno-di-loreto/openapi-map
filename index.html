OpenAPI Map

The OpenAPI Map (fka. OpenAPI Specification Visual Document) aims to help you find your way in the OpenAPI Specification (fka. Swagger Specification) documentation.
It has been created by Arnaud Lauret, the API Handyman .

This tool now shows the 3.0 version (3.0.1 precisely) of the OpenAPI specification with a complete change log from version 2.0. You can select the version (2.0 or 3.0) by using the links on the top right navigation bar and also use direct links to each version:
  • http://openapi-map.apihandyman.io?version=2.0
  • http://openapi-map.apihandyman.io?version=3.0

Controls

Mouse scroll up to zoom in
Mouse scroll down to zoom out
Mouse drag to move viewpoint
Move mouse over node to show its documentation
Click on node to expand or collapse it
